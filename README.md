## Concentration小游戏

整体采用MVVM架构，使用SwiftUI编写的益智小游戏，十分适宜用来练手，代码不复杂。非常适合体会MVVM架构的好处

### 游戏玩法

此项目是一个记忆力小游戏，简而言之就是翻开卡牌，并记住其内容，当翻开的两张卡牌内容相同时，即可消去。每消去一对卡牌加十分，每次翻开卡牌扣一分，最终会记录最终得分，并有排行榜功能
